java extension pack をインストール。
Tomcatは主にJavaのウェブアプリケーションを実行するためのサーバー であり、コンソールアプリケーションには必要ありません。
とのことでひとまず１つのゲームを作成する。
NumberGuessingGame。ひとまずコピペになる。
ファイルを保存（これがうまくいかないと以下うまくいかない）
javac NumberGuessingGame.javaでコンパイル。
コンパイルが成功するとjavac NumberGuessingGame.classが作成される！
java NumberGuessingGameで実行。
