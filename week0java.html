1行目に出てくる、packageという記述は、パッケージ。 このプログラムを分類する「入れ物」の名前を決める部分。 ここでは、day1というパッケージで同一パッケージ内に作成 されるプログラムは互いに関係性が深いもものとされる。
パッケージの設定を package day1; とすることにより、このプログラムはday1と呼ばれるパッケージ に属することが示される。

classという名前の後に、Sample101。これはクラス名。 プログラムのまとまった処理の単位。 同一パッケージないには、同じ名前のクラスが存在してはいけない。 パッケージが異なれば、同じクラス名が存在してOK。

public static void main(…)の部分、メインメソッドの宣言と呼ばれるところ。 Java言語は、原則的にこの中に処理を書くことにより、プログラムが実行される。 処理の中身は、"{"および"}"で囲まれている。
この処理の中には、System.out.println()という命令があり、 これは、コンソールに文字を出力するもの。 「main()内に処理を書けば、実行される」

System.out.println()は ()で囲まれたものを表示する命令。 文字列の場合は、"で挟む。文字列でなければ不要

printは改行しない。 printlnは出力後改行をはさむ。

練習問題で：実行結果は同じでも、処理が一行しかないような形に プログラムを変更しなさい。(ヒント：エスケープシーケンスを利用する）

int a; // 変数の宣言
int b = 3; // 初期化と代入を同時に行う。
int add,sub; // 複数の変数を同時に宣言
double avg; // int以外の変数を宣言
a = 6; // 代入（最初に値を入れるので、”初期化”と言う。
add = a + b; // a,bの和を求める。
sub = a - b; // a,bの差を求める。
avg = (a + b) / 2.0; // a,bの平均値を求める。
System.out.println(a + " + " + b + " = " + add)
System.out.println(a + " - " + b + " = " + sub);
System.out.println(a + "と" + b + "の平均値：" + avg);

変数に値を入れることを代入と言う
代入演算子：短くても同じ短く計算
int a;
double b,c,d;
a = (int)1.23; // キャストで代入
b = 1.23;
c = 10; // キャストなしで代入
d = (double)c; // キャストありで代入
System.out.println("a=" + a + " b=" + b + " c=" + c + " d=" + d);
＞a=1 b=1.23 c=10.0 d=10.0
型変換がおこなわれている。
頭に変換したい値の型を先頭につけることを、「キャスト」と言う。
aはint型なので少数を代入しようとするとエラーになる。
dはdouble型なので、整数を代入しようとしたところでエラーにはならず表示される。

public static void main(String[] args) {
	int dice = (int)(Math.random()*7)+1;    //  1から7までの乱数を発生させる
	//  サイコロの目を入力
	System.out.println("さいころの目:"+dice);
	//  値が、サイコロの目の範囲内かどうかを調べる
	if(1 <= dice && dice <= 6){
		//  さいころの目が、偶数か、奇数かで、処理を分ける。
		if(dice == 2 || dice == 4 || dice == 6){
			System.out.println("丁（チョウ）です。");  //  偶数ならば丁（チョウ）
		}else{
			System.out.println("半（ハン）です。");   //  奇数ならば半（ハン）
		}
	}else{
		System.out.println("範囲外の数値です。");
	}
}

乱数
0からn-1までの乱数を発生させる方法
(int)(Math.random()*n);
続いて、1からnまでの乱数を発生するためには、以下のようにします。
(int)(Math.random()*n)+1;
